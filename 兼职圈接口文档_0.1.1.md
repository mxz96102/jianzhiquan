## 一.客户端部分
1.用户登录接口
http://HOST/auth/login

* METHOD：POST
* PARAM：


    String phonenum:手机号
    String credential:密码
* SUCCESS：


    {
        "msg":"SUCCESS",
        "code":"200",
        "result":null    
    }
* FAIL：


    {
        "msg":"wrong password or user not exist, check your password",
        "code":"403",
        "result":null
    }
* 备注：


    除 /auth/** 接口外，所有接口需登录后调用
